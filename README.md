# Brain Tumor Segmentation Web App

This project demonstrates a UNet-based deep learning model for brain tumor
segmentation using 2D MRI slices.

## Features
- Grayscale MRI input
- UNet segmentation model
- Preprocessing and post-processing pipeline
- Streamlit-based web interface
- Local and cloud deployment ready

⚠️ This application is for academic and research purposes only.
Not intended for clinical use.
